# CS Study

## C 프로그래밍
 * (연구실 활동) C프로그래밍 교재의 모든 문제풀이를 실습하고 소스를 저장하였다.

## 자료구조
 * (연구실 활동) 자료구조를 공부하고 응용하여 토이프로젝트를 작성하였다.

 * 트리를 이용한 계산기 구현
 * 링크드리스트와 그래프를 활용한 미로찾기 구현
